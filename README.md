## Futebol Data / CesarScore
Esse projeto consiste em um site de palpites e acompanhamento de estatísticas da Champions League. Ele permite a visualização da tabela atualizada, os placares das partidas e os artilheiros da competição. Além disso, o usuário pode se cadastrar para criar e editar palpites sobre os resultados das próximas rodadas.


![Imagem da home page](/public/arena-bg.png)

### Tecnologias Utilizadas
O projeto foi desenvolvido inteiramente a partir do framework Next.js, e os dados são da API futebol - https://www.api-futebol.com.br/
